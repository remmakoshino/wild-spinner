# Stage 3/3 提案と実装メモ

## 提案

Stage 3/3 は、既存の氷結帯・崩落床に加えて「最終防衛ライン」を導入し、
ゲーム終盤に向けた緊張感と難易度の段階上昇を明確にする。

- ゴール手前で封鎖イベント発生
- 増援敵を全滅するとゴール解放
- 封鎖中は BGM を boss セクションへ固定して圧力を演出

## 実装済み内容

- Stage 3 定義に finalChallenge を追加
- triggerX 到達で shooter + support 増援スポーン
- 増援を含む全敵撃破でゴール解放
- ゴール封鎖中は赤ティント、解放後は黄ティント
- HUDメッセージで進行条件を案内

## 検証項目

- triggerX 到達でイベントが1回だけ発火する
- 封鎖中はゴールに触れてもクリア遷移しない
- 全敵撃破でクリア可能になる
- Stage 3 テストで finalChallenge 定義を検証する
